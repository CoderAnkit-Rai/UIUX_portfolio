# UI/UX Portfolio - Hospital Management System

A responsive portfolio website showcasing a comprehensive Hospital Management System design project.

## Features

### Portfolio Sections

1. **Introduction Section**
   - Personal introduction and background
   - Skills and expertise display
   - Professional profile

2. **Hospital Management System**
   - Interactive demo with full functionality:
     - **Patient Management**: Add and remove patients
     - **Appointment Scheduling**: Schedule appointments with doctors
     - **Device Management**: Track medical devices and their status
     - **Alert System**: Real-time alerts for non-functional devices

3. **User Personas**
   - Two detailed user personas with images
   - User needs and pain points

4. **Heuristic Evaluation Report**
   - Comprehensive evaluation based on Nielsen's 10 Usability Heuristics
   - Priority ratings for each heuristic

5. **Low Fidelity Wireframes**
   - Initial wireframes
   - Information Architecture diagram
   - Empathy Mapping visualization

6. **High Fidelity Wireframes**
   - Refined wireframes with detailed UI elements
   - Color schemes and typography

## Technologies Used

- HTML5
- CSS3 (with CSS Grid and Flexbox)
- JavaScript (Vanilla JS)
- Responsive Design (Mobile-first approach)

## Getting Started

1. Open `index.html` in a web browser
2. No build process or dependencies required
3. All functionality works out of the box

## Customization

### Adding Your Images

Replace the placeholder divs with your actual images:

1. **User Personas**: Add images in the `.persona-image` divs
2. **Wireframes**: Add images in the `.wireframe-placeholder` divs
3. **Information Architecture**: Add your IA diagram
4. **Empathy Mapping**: Add your empathy map visualization

### Updating Content

- Edit the introduction text in the `#intro` section
- Update persona information in the `#personas` section
- Modify heuristic evaluation details in the `#heuristic` section
- Add your wireframe images to the respective sections
